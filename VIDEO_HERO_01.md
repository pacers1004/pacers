## 제작 순서 (대표님 실행)
1. 아래 **클립 12개**를 힉스필드 image-to-video로 생성 (소스 스틸 → 모션 프롬프트)
2. **내레이션 스크립트**를 TTS(TypeCast 한국어 감정보이스 권장)로 음성 생성
3. 자막 + 음성 + 클립을 CapCut에서 조립 (자막 1줄 = 클립 1개 = 음성 1문장)
4. 채널별 재단 (§MARKETING.md §10-D) → 발행
